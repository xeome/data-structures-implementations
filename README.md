# Data Structures Library

This repository contains a collection of various data structures implemented in C, including:

- Hashmap
- Heap Tree
- Huffman Tree
- Linked List
- Queue
- Vector

These implementations are intended for learning and experimentation purposes, and are not recommended for use in production environments.

## Installation

To use the data structures in this repository, clone the repository and include the relevant source files in your project.

## Usage

Each data structure has its own set of functions for interacting with it. Please refer to the documentation within the source files for more information on how to use each data structure.

## Examples

Examples of how to use each data structure can be found within the main function of the corresponding source file.
